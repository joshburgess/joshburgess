## Recent Projects

### Written in Rust

- [typeway](https://github.com/joshburgess/typeway): type-level web framework where the API is one type, deriving servers, clients, and OpenAPI schemas
- [resolute, pg-wired, & pg-pool](https://github.com/joshburgess/resolute): ground-up PostgreSQL client stack with compile-time checked queries, binary wire protocol, and async pool
- [proptest-lockstep](https://github.com/joshburgess/proptest-lockstep): lockstep-style stateful property testing, with 236 machine-checked theorems in Lean 4
- [reify-reflect](https://github.com/joshburgess/reify-reflect): safe ecosystem of crates for moving values between the type level and the value level
- [elm-ast](https://github.com/joshburgess/elm-ast): library for parsing and constructing Elm 0.19.1 ASTs, inspired by syn
- [elm-assist](https://github.com/joshburgess/elm-assist): native-Rust Elm tooling: linter, dead code analysis, refactoring, code search, TUI, and LSP server
- [elm-client-gen](https://github.com/joshburgess/elm-client-gen): generates Elm types, decoders, encoders, and HTTP clients from Rust types and Axum handlers
- [mythic](https://github.com/joshburgess/mythic): fast, batteries-included static site generator
- [hoogle-tui](https://github.com/joshburgess/hoogle-tui): terminal UI for Hoogle, Haskell's type-aware search engine
- [cabalist](https://github.com/joshburgess/cabalist): opinionated toolkit for managing Haskell `.cabal` files, with a TUI, scriptable CLI, and LSP server

### Written in Haskell

- [valiant & pg-wire](https://github.com/joshburgess/valiant): ground-up PostgreSQL client stack for Haskell with compile-time checked queries and binary wire protocol
- [hedgehog-lockstep](https://github.com/joshburgess/hedgehog-lockstep): lockstep-style stateful property testing on top of Hedgehog's state machine framework

- [tagged-ts](https://github.com/joshburgess/tagged-ts): tagged unions code generation library for discriminating tastes
